# Projet 1 - Analyse des Données de Construction

**Thématique :** Optimisation des coûts et de la durée des chantiers

## Structure du projet

```
Projet 1 - Analyse des Données de Construction/
├── Construction_Projects.csv      # Dataset (1000 projets)
├── construction_analysis.py       # Script principal Python
├── requirements.txt
├── RAPPORT_ANALYSE_CONSTRUCTION.md
├── RAPPORT_ANALYSE_CONSTRUCTION.html
├── outputs/                       # Graphiques et données
│   ├── 01_distribution_couts.png
│   ├── 02_cout_vs_duree.png
│   ├── ...
│   ├── Construction_Projects_Analytics.xlsx
│   └── metrics_summary.txt
├── sql/
│   ├── 01_create_table.sql
│   ├── 02_data_preparation.sql
│   └── 03_analytical_queries.sql
└── power_bi/
    ├── Power_Query_Transformations.pq
    └── Instructions_Power_BI.md
```

## Installation et exécution

1. Installer les dépendances :
   ```
   pip install -r requirements.txt
   ```

2. Exécuter l'analyse :
   ```
   python construction_analysis.py
   ```

3. Consulter le rapport : ouvrir `RAPPORT_ANALYSE_CONSTRUCTION.html` dans un navigateur
